# Fastget

Rapid order fulfillment system. Built with Next.js, designed for speed.

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Open http://localhost:3000
```

## Tech Stack

- **Framework:** Next.js 14 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Database:** Google Sheets (MVP) → PostgreSQL (future)

## Development

This project uses gstack AI workflow skills. Available commands:

- `/office-hours` - Product planning and ideation
- `/plan-ceo-review` - Strategic review
- `/plan-eng-review` - Architecture review
- `/ship` - Create PRs and deploy
- `/qa` - Test the application
